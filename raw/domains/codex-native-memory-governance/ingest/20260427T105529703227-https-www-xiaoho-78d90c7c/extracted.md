Codex的自我进化

Codex APP的官方示例里面有一段非常有用的Case，是使用 自动化 的功能来让Codex自我改进Skills，这其实也是当前所有具有自我进化功能的Agent的核心关键。
具体操作如下：
1. 复制以下提示词：
```
Scan all of the `~/.codex/sessions` files from the past day and if there have been any issues using particular skills, update the skills to be more helpful. Personal skills only, no repo skills.
If there’s anything we’ve been doing often and struggle with that we should save as a skill to speed up future work, let’s do it.
Definitely don't feel like you need to update any- only if there's a good reason!
Let me know if you make any.
```
2. 打开Codex App ，创建一个空的项目（或者已经存在的项目也可以，这是指定一个运行的目录）
3. 打开 Automations（自动化）功能，创建 新的自动化
4. 粘贴以上提示词内容， 然后随便起个名字， 选择刚才创建的空的项目，模型直接 gpt5.4 xhigh 拉满
5. 选择一个 不在工作时间的时间段（但要注意是 电脑没关机的时候）
6. 最后点击 创建 即可！
这个自动化的任务会在指定的时间 扫描过去一天的会话文件的，看看是否有可以总结为skill的操作流程，然后创建出新的Skill，后面再使用的时候就会自动调用了。
如果默认的权限不是“完全允许”或者“自动审核”的话，需要注意运行后确认权限哦。
